set_required_cookie()

set_required_cookie(string $user_login, string $user_role, int $user_id, int $expire) : array

Called by ClipIt (authentication master), to set an authentication cookie.

Parameters

string $user_login

The user's login ID, sometimes called a "username".
string $user_role

An identifying string for the user's role.
int $user_id

An optional numeric ID, probably from ClipIt's database,
int $expire

The time the cookie expires.

Returns

array —

Debug information, including input parameters.

Tags

example

= $auth->set_required_cookie( 'pebs123', 'teacher', 999 );

      var_dump( $result[ 'user_role' ] );

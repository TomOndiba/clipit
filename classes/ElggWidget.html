__construct()

__construct(mixed  $guid = null)

Load or create a new ElggObject.

If no arguments are passed, create a new entity.

If an argument is passed, attempt to load a full ElggObject entity. Arguments can be:

  • The GUID of an object entity.
  • A DB result object from the entities table with a guid property

Parameters

mixed $guid

If an int, load that GUID. If a db row, then will attempt to load the rest of the data.

Throws

\IOException

If passed an incorrect guid

\InvalidParameterException

If passed an Elgg* Entity that isn't an ElggObject

__clone()

__clone() : void

Clone an entity

Resets the guid so that the entity can be saved as a distinct entity from the original. Creation time will be set when this new entity is saved. The owner and container guids come from the original entity. The clone method copies metadata but does not copy annotations or private settings.

Inherited from
\ElggEntity

Tags

note

metadata will have its owner and access id set when the entity is saved and it will be the same as that of the entity.
